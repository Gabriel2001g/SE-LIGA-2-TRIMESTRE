Qual a diferença entre margin e padding?

Margin é o espaço fora de um elemento, enquanto padding é o espaço dentro do elemento, entre seu conteúdo e sua borda.

O que é JavaScript?

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web.
